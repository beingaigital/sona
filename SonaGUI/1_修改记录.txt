--------------------------------------------------------------------------------
五、如何本地验证（给老师/助教）
--------------------------------------------------------------------------------
1. 项目根：pip install -r requirements.txt && pip install -e .
2. sona serve --host 127.0.0.1 --port 8765
3. 浏览器打开 http://127.0.0.1:8765/docs
4. GET /health 应返回 {"status":"ok",...}
5. POST /v1/analyze-event 会跑完整事件分析，耗时长，验收可仅测 /health 与 404 任务查询
