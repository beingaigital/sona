## 6. 「自动召回」验证说明（健康）

配置文件：

```text
workflow/domain_routing.json
```

「健康」域下包含 **关键词匹配**、`must_include`（如 `playbook_health.md`、`concepts/医疗纠纷.md`、`concepts/食品安全.md`、若干 `cases/`）、`prefer`（如 `公共卫生` 概念、`霸王茶姬` 实体）及注入上限等。

**验证方式（示例）**：在项目根、依赖安装完备的环境下执行：

```bash
python -c "
from pathlib import Path
from workflow.wiki_cli import answer_wiki_query
root = Path('.').resolve()
out = answer_wiki_query('宁波妇儿医院医疗纠纷舆情应对', topk=9, style='teach', project_root=root)
meta = out.get('_wiki_meta') or {}
print(meta.get('domain'))
for s in (out.get('sources') or [])[:10]:
    print(s.get('path'))
"
```

**预期**：`_wiki_meta.domain` 为 **`健康`**；来源路径中出现 **`playbook_health.md`**、**`concepts/医疗纠纷.md`** / **`食品安全.md`** 及 **`cases/H01_...`** 等之一即可作为「健康事件可召回本域知识」的举证。
