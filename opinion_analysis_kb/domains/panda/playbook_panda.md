# 大熊猫舆情 Playbook

本 Playbook 面向**舆情分析**：生物学事实以 `docs/` 内基地科普与《大熊猫谣言与辟谣》为准；报道级事实须单独取证，禁止用科普文本替代新闻核查。

## 1. 议题地图

- **保护伦理**：圈养福利、行为富化、疾病与死亡披露节奏。
- **文旅与商业**：门票、排队、IP 联名、过度「萌化」营销。
- **粉丝与圈层**：个体熊猫「饭圈化」、拉踩、举报战、跨平台搬运。
- **国际与外交叙事**：租借/旅居、外媒框架与国内情绪的错位。
- **谣言与伪科普**：剪辑、断章取义、张冠李戴（把其他熊科/小熊猫事件嫁接到大熊猫）。

## 2. 独特传播机制

| 机制 | 说明 |
|------|------|
| 情感极化 | 高萌内容易触发「护崽」道德情绪，事实争议被压缩为「站队」。 |
| 可视化证据霸权 | 短视频片段被视为「亲眼所见」，完整监控/兽医记录往往滞后。 |
| 拟人化投射 | 命名、CP、生日应援等将动物人格化，放大误解（如将正常争斗解读为虐待）。 |
| 跨境框架差 | 同一事件在国内「管理失职」与海外「政治符号」叙事可能并行。 |

## 3. 风险模式（RiskPattern）

- **信息披露真空**：官方沉默 → 阴谋论填空。
- **专业话语缺位**：仅情绪回应 → 科普账号与谣言对冲失败。
- **饭圈化攻击**：举报、挖坟、牵连无关饲养员或合作品牌。
- **直播与剪辑伦理**：夜间镜头、配乐、慢放制造「受苦」错觉。
- **次生舆情**：删帖、律师函、怼粉丝触发平台治理话题。

## 4. 回应策略（ResponseTactic）

1. **事实三段式**：可公开事实（时间/地点/处置）＋暂不可公开原因（法律或生物福利）＋下一步核查节点。
2. **兽医/行为专家背书**：解释行为谱系（进食、争斗、育幼），对齐 `docs/【熊猫知识】行为特点.md` 等。
3. **谣言对表**：直接引用 `大熊猫谣言与辟谣：全面知识库.md` 中条目，附可追溯来源。
4. **平台协同**：对恶意剪辑提供时间轴对照素材（在合规前提下）。
5. **文旅侧**：区分「动物福利」与「游客体验」，避免混谈引发公平感争议。

## 5. 检索关键词（RAG）

## 6. 证据层级

1. 监管/合作机构通报  
2. 园区/中心官方说明  
3. 同行评议与权威科普（本包 `docs/`）  
4. 媒体报道（需交叉验证）
