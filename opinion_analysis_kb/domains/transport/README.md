# 交通舆情领域包（SONA 垂类升级）

## 资料构成

1. **课题摘录型领域文档（30 篇）**  
   路径：`materials/交通舆情课题摘录/`。内容依据小组提供的《中国交通舆情发展报告（2021–2025）》蓝皮书汇总稿及分报告 **B1、B5、B9、B10、B11** 等整理为可检索 Markdown（要点 + 归纳），**正式引用请以原 Word/PDF 为准**。

2. **Playbook**  
   `playbook_transport.md`：面向分析报告写作的处置建议、监管口径与检索关键词。

3. **案例（10 个）**  
   `cases/T01–T10.md`：覆盖铁路/高铁服务与冲突、12306 与票务公平、航空服务与重大事件长尾、网约车枢纽治理、公交与城市治理、道路安全与智驾、水路港口安全、政策谣言、危机沟通「事心双解」等。
