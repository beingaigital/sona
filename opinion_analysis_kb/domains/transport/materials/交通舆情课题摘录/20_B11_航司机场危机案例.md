# B11 摘编：航空公司与机场危机案例启示

**来源**：B11 航司、机场排行段落。**正式引用以原稿为准。**

## 要点

- **东航**：重大危机中 **快速发声 + 高管致歉 + 家属援助机制** 体现系统化应对（报告叙述）。
- **国航**：起落架安全销未拆等事件中 **企业账号首发 + 大V协同 + 深表歉意与追责承诺**，生存时间短。
- **南航**：特殊旅客事件澄清事实同时给出 **老人致歉与人文关怀表述**。
- **国泰**：歧视事件中 **致歉、调查、解聘** 组合拳 + 大V矩阵传播。
- **机场**：上海机场集团事实澄清快（辟谣「取消率 72%」、网约车政策细则）；白云机场雕塑争议 **道歉时机与周全度不足** 触发第二轮性别议题；宝安机场、萧山机场等案例显示 **员工安全与旅客焦虑** 均需纳入声明。

## 监管口径归纳

机场声明忌 **「调查中」单打**——须并行 **旅客安置/改签路径 + 对员工伤亡的关切（如涉及）**。
